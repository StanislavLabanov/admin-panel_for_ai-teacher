**Admin Panel for AI Teacher** is an administrative panel for managing the "AI Teacher" system. The project was developed to simplify the work of administrators and teams managing the AI-powered educational platform.

## Key Features
- User management (administrators, teachers, students)
- Test setup and creation
- View system usage analytics: user activity, progress, statistics
- Interface for managing access rights and roles
- Interaction with the backend via REST API

## Technologies
- **Frontend:** TypeScript, Vite, HTML/CSS
- **Build Tool:** Vite
- **Project Structure:** SPA with a modern approach to modularity and components

## Advantages
- High performance thanks to Vite and modern web technologies
- Scalability: easily add new modules and expand functionality
- Ease of administration: centralized interface for managing users and content
- Potential for integration with AI services

## Possible Use Cases
- Educational platforms using AI to create learning tasks and provide feedback
- EdTech startups with the need to manage users, courses, and content
- Corporate training, where administrators control the roles of teachers and students

## Installation and Launch
1. Clone the repository:
git clone https://github.com/StanislavLabanov/admin-panel_for_ai-teacher.git
2. Go to the project folder:
cd admin-panel_for_ai-teacher
3. Install dependencies:
yarn
4. Run the dev server:
yarn dev
